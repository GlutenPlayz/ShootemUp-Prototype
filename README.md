# Shoot 'em up Prototype

**Engine:** Unity  
**Language:** C#  
**Project Type:** Gameplay Prototype  

## Overview

Small gameplay prototype inspired by classic arcade fixed shooters like Galaga.

The goal of the project was to explore the Unity engine and practice object oriented programming through a game development project.

## Gameplay Features

- Player-controlled ship with free movement  
- Projectile-based shooting system  
- Enemy spawning and random shooting intervals
- Collision and destruction handling w/ animated VFX
- Support for alternate control schemes

## Systems Implemented

### Player Controller
Handles movement and shooting input while maintaining responsive controls and consistent gameplay feel with the use of delta time.

### Projectile System
Reusable projectile logic for handling movement, collision detection, and damage interactions.

### Enemy Spawning
Enemy wave spawning system with random variables for a more dynamic gameplay experience.

## Development Goals

- Become more familiar with Object Oriented Programming in a game development scenario
- Recreate the core gameplay loop of classic arcade shooters  
- Maintain responsive and readable gameplay  
- Explore how enemy spawn patterns affect difficulty and pacing
- Experiment with different overall game pacing and explore how it affects game feel

## What I Learned

Through this prototype I explored how enemy spawning patterns and projectile density influence difficulty and player decision-making.  
The project also helped me practice structuring gameplay scripts and building reusable systems in Unity.
